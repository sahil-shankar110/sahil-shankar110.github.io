01 — About

A bit
about me.

I study Artificial Intelligence at Mehran University of Engineering & Technology, graduating in 2028. My CGPA is 3.35 out of 4.0. Most of my time outside class goes into building and deploying real projects — not writing course assignments.

Generative AI · NLP · Live App
Natural Language Data Analytics Engine

Type a question in plain English, get back a SQL query, a results table, and a chart — automatically. Built specifically for non-technical users: small business owners, analysts, and domain experts who have data but cannot write SQL. Powered by LLaMA 4 Scout via Groq API for fast, schema-aware query generation.

  • Natural language to SQL with schema-aware prompting
  • Supports CSV, Excel, JSON, and SQLite file uploads
  • Session history stores the last 10 queries
  • Read-only SQLite with SELECT-only enforcement at two independent layers
  • Auto chart selection — bar, line, pie, scatter via Plotly

Agentic AI · Tool Calling · Live App
Agentic Chatbot with Web Search

A conversational AI system built on LangChain's tool-calling architecture with real-time web access via Tavily API. The user sets a system prompt, selects a model — Gemini or LLaMA — and toggles web search on or off. The agent retrieves live internet data when needed and answers with verified, up-to-date information. Backend runs on FastAPI deployed on Render; frontend on Streamlit Cloud.

  • Tool-calling agent architecture with conditional web retrieval
  • Tavily API integration for real-time internet search
  • Dual LLM support: Gemini and LLaMA selectable per session
  • User-defined system prompt for full control over AI behavior
  • Query history stores the last 5 conversations
  • Decoupled FastAPI backend and Streamlit frontend deployment

LLM Automation · ChromaDB · Live App
AI Job Email Generator

Paste a job posting URL, get a professional, personalized outreach email in seconds. Built for job seekers, internship applicants, and freelancers who waste hours writing and rewriting the same email for different job descriptions. The application scrapes the job posting, extracts role requirements using prompt engineering, retrieves matching projects from a ChromaDB vector store, and generates a tailored email via LangChain and MetaLlama.

  • Web loader extracts job requirements from any posted URL
  • ChromaDB vector store retrieves semantically matched sample projects
  • MetaLlama via LangChain generates the personalized email
  • Clean Streamlit UI — name, email, URL, and the email is ready

Agentic AI · LangGraph · RAG
Agentic Chatbot Systems (LangGraph Collection)

A collection of chatbots built with LangGraph — not standard Q&A bots. These are state machines where the LLM decides which tools to invoke, processes the results, and determines the next step without hardcoded branching logic. The collection covers RAG with both ChromaDB and FAISS retrieval, MCP tool integration for real-world action execution, and a reasoning chatbot that evaluates its own responses through a built-in rating system.

  • LangGraph state machines with conditional routing and tool selection
  • RAG pipelines using ChromaDB and FAISS vector retrieval
  • MCP tool integration for external action execution
  • Reasoning chatbot with self-evaluation and rating logic
